# Таблица расчета апгрейда зданий Bastion Siege

Расчет эффективности апгрейда зданий основан на понятии <a href="https://ru.wikipedia.org/wiki/%D0%A1%D1%80%D0%BE%D0%BA_%D0%BE%D0%BA%D1%83%D0%BF%D0%B0%D0%B5%D0%BC%D0%BE%D1%81%D1%82%D0%B8">Срок окупаемости</a>.
Т.е. периода, за который вложенные средства окупятся и начнут приносить доход. Чем короче этот период - тем выгодней инвестиция.
Соответственно, чем меньше число в графе "Окупаемость апгрейда в часах", тем выгоднее апгрейд здания. Самые выгодняе для апгрейда здания выделяются зеленым цветом, самые невыгодные - красным.
